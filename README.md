# Baze de Date

Pagina tutoriatului de Baze de Date @ FMI 2025-2026.

Cum salvați mai ușor toate fișierele din repository *(din orice terminal)*:
```
cd path/to/local_dir/
git clone https://github.com/mateipiratul/tutoriat_bd_2526
rm .git
cd tutoriat_bd_2526
ls
```

| **Fișier/Folder** | **Descriere** |
|--------|-----------|
| `cursuri/` | Cursurile doamnei lector doctorand Cezara Benegui |
| `laboratoare/` | Exerciții de laborator în SQL |
| `tutoriate/` | Materiale redactate din anul universitar 2024/2025 |
| `documentatie_sql.pdf` | Oracle Database SQL Language Reference |
| `diagrama_hr.pdf` | Diagrama E/R de referință a cursului |
| `tema_curs/` | Exemplu de temă de curs *(specific seriei 15)* |
| `tema_laborator/` | Exemplu de temă de laborator *(specific seriei 15)* |
